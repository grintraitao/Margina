Rounded selector pill from the wallet header ("All Networks", "All accounts"). Shows overlapping chain dots, a label, and a chevron.

```jsx
<NetworkPill label="All Networks" dots={['#627EEA','#9945FF','#00b3ED']} />
<NetworkPill label="All accounts" dots={['#ff4070','#00CCC0']} active />
```

Border turns Blue Sky on hover/active. Pass up to 3 `dots` colors.
